One Command Setup

Add Huginn to any WhatsApp group with a single command. No registration needed.

Three steps is all you need.

01

Save the number

Save Huginn's WhatsApp number as a contact on your phone. That's the only thing you do outside WhatsApp.

02

Add it to your group

Open any WhatsApp group you're already in. Tap Add Participant, search Huginn, done. It joins silently.

03

Send one command

Type /follow Brazil in the group. From that moment, every goal, red card, and odds shift fires automatically straight into the chat.
